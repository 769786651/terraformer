(static) arcgisToGeoJSON(JSON, idAttributeopt) → {object}

Converts [ArcGIS JSON](https://developers.arcgis.com/documentation/core-concepts/features-and-geometries/) into GeoJSON.
Parameters:
Name Type Attributes Description
JSON object The input ArcGIS geometry, feature or feature collection.
idAttribute string <optional>
 When converting an ArcGIS Feature its attributes will contain the ID of the feature. If something other than OBJECTID or FID stores the ID, you should pass through the fieldname explicitly.
Source:
  • arcgis/index.js, line 29
Returns:
GeoJSON. ```js import { arcgisToGeoJSON } from "@terraformer/arcgis" arcgisToGeoJSON({ "x":-122.6764, "y":45.5165, "spatialReference": { "wkid": 4326 } }); >> { "type": "Point", "coordinates": [ -122.6764, 45.5165 ] } ```
